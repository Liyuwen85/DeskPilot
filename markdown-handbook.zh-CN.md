# Markdown 手册

这是一份适合 DeskPilot 使用的常见 Markdown 语法速查表。

## 标题

```md
# 一级标题
## 二级标题
### 三级标题
```

## 段落与换行

普通文本直接换行还是同一段。

如果要换行：

```md
第一行  
第二行
```

或者空一行开始新段落。

## 强调

```md
**加粗**
*斜体*
~~删除线~~
`行内代码`
```

## 列表

无序列表：

```md
- 项目一
- 项目二
  - 子项目
```

有序列表：

```md
1. 第一项
2. 第二项
3. 第三项
```

任务列表：

```md
- [x] 已完成
- [ ] 待处理
```

## 引用

```md
> 这是一段引用
>> 这是嵌套引用
```

## 链接与图片

链接：

```md
[DeskPilot](https://github.com/Liyuwen85/DeskPilot)
```

图片：

```md
![说明文字](./image.png)
```

## 代码块

```md
```ts
console.log("Hello DeskPilot");
```
```

可以在开头标明语言，如 `ts`、`js`、`python`、`bash`。

## 分割线

```md
---
```

## 表格

```md
| 列 1 | 列 2 |
| --- | --- |
| A | B |
| C | D |
```

## 数学公式

行内公式：

```md
$E = mc^2$
```

块级公式：

```md
$$
\int_a^b f(x)\,dx
$$
```

## 建议

- 标题层级尽量连续，不要从 `#` 直接跳到 `####`
- 代码块优先标注语言，便于高亮
- 图片尽量使用相对路径，便于项目迁移
- 长文档建议配合大纲使用
